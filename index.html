Spanish Territory Scraper

Instructions

To use, input at least the street and Zip Code, otherwise nothing will be returned. You can omit the house number, in which case probably the lowest house number will be evaluated. If you specify the house number, you can do so using just a single address, or a range of addresses. To specify a range of addresses, do so using brackets. For example, if you want to search between the addresses of 10 and 20, you would put the following into the house section: [10-20]

If the tool is successful, it will send an Excel spreadsheet to your browser as a download. This will probably appear as a download dialog, but it depends partly on how your browser is set up. The spreadsheet will have two tabs; the first is a pretty version that contains only the residents who are likely spanish due to the spelling of thier last names. The second tab is some of the raw information that had to be evaluated and is useful for reference.

If the tool is unsuccessful, well you'll probably see some error message or other junk in the Excel file. I really don't know. But I will say that if you don't see anything in the first tab, it could mean that the program worked, but didn't find any names that were likely spanish.
